# @bluzzi/lingua
The WebAssembly compiled version of [Lingua Rust](https://github.com/pemistahl/lingua-rs).

## Installation
```sh
npm install @bluzzi/lingua
```

```sh
yarn add @bluzzi/lingua
```

```sh
pnpm install @bluzzi/lingua
```
